Vertex:
    - Usage: v <x> <y> <z>
    - Description: Specifies a vertex position

Non-geometry vertex:
    - Usage: ng <x> <y> <z>
    - Description: Specifies a vector value not used for vertex geometry

Face:
    - Usage: f <v1> <v2> <v3>
    - Description: Specifies a face

Camera:
    - Usage: camera <position v> <look ng> <up ng> <hangle>
    - Description: Specifies the camera

Sphere:
    - Usage: sphere <position v> <radius>
    - Description: Specifies a sphere

Directional light:
    - Usage: ld <direction ng> <color wavelength start> <color wavelength end> <function points>
    - Description: Specifies an directional light

Material library:
    - Usage: mtllib <relative path>
    - Description: Imports material defines from a separate file

Set Material:
    - Usage: usemtl <material name>
    - Description: All geometry following this declaration will use the material defined by <material name>

Area Light:
    - Usage: la <position v> <color wavelength start> <color wavelength end> <f1> <f2> <f3> <f4> <f5> <f6> <function points>
    - Description: Specifies an area light

Sphere Light:
    - Usage: lsphere <position v> <color wavelength start> <color wavelength end> <radius> <function points>
    - Description: Specifies a sphere area light
